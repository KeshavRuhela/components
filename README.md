# Component Effects

## Folder naming

Based on BEM:

- Block: Subject.
- Element: Verb.
- Modifier: Object.

Example:

- subject\_\_verb_object
- block-name\_\_element-name_modifier-name

## Folders

- /applied: effect applied on examples but not ready to production;
- /src: source of effect ready to production
- /dist: effect ready to production
