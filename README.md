# Component Effects

## Folder naming

1. Subject
2. Verb
3. Object

Example:

- subject-verb-object
- block-name-element-name-modifier-name
- any-hover-show

## Folders

- /applied: effect applied on examples but not ready to production;
- /src: source of effect ready to production
- /dist: effect ready to production
